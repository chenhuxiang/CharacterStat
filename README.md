# CharacterStat
CharacterStat字数统计工具，绿色免安装的单文件，快速统计字符总数、字数总数、中文字符、中文标点、英文标点、数字字符、英文字符、英文单词、段落数量。

## 功能说明：
1. 输入文字内容，立即统计字符总数、中文数量等
2. 文本中英文标点符号互转
3. 全局快捷键F2, 一键统计其他程序选中的文本字数
4. 词频统计结果导出

## 计算公式说明：
一个汉字占两个字符，
字数总数=中文字符 + 中文标点 + 英文标点 + 数字字符 + 英文字符 + 空格等其他符号。
